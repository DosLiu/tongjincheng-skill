## 这是什么

这是一个偏实战的恋爱与社交辅助 skill，不是单纯模仿某个主播说话。

它主打：
- 从刚加微信到邀约的全流程辅助
- 根据聊天记录判断女生当前兴趣
- 直接生成 1-3 条可发送回复
- 帮你把握节奏，避免低价值追求
- 遇到明确拒绝时及时止损

它适合：
- 追女生
- 追女朋友
- 推进暧昧
- 帮忙回微信
- 判断还有没有戏
- 从聊天过渡到线下邀约

---

## 核心特点

- 会判断阶段：刚加上、破冰期、熟悉期、升温期、邀约窗口、冷淡期、拒绝期
- 会识别信号：绿灯、黄灯、红灯
- 会控节奏：知道该续聊、试探、拉近、邀约，还是收一收
- 会生成回复：每次给 1-3 条可直接发送的话
- 会止损：明确拒绝或连续冷淡时，不鼓励纠缠

---

## 风格定位

这个 skill 不是舔狗风，也不是油腻情圣风。

它想做的是：
- 真诚
- 松弛
- 有主导感
- 有一点坏坏的吸引力
- 但不搞羞辱、操控、驯化

可以有一点“渣感外壳”：
- 不跪
- 不慌
- 不把喜欢一次性给满
- 有边界
- 会带节奏

但不会提供这些内容：
- PUA
- 打压
- 骚扰
- 情绪操控
- “打开女性奴性”之类的话术

## 输出方式

看到聊天记录后，优先输出：

1. 现在处于什么阶段
2. 女生是绿灯、黄灯还是红灯
3. 这一步最该做什么
4. 1-3 条可直接发送的回复
5. 发哪条最稳，什么时候发
6. 这一步最该避免什么

## 例子

### 刚加微信

用户：

```text
昨天线下认识加了微信，今天第一句怎么发？
```

Skill：

```text
阶段：刚加上
信号：信息不足，先轻开场
目标：续上记忆点，不要尬聊

可直接发：
1. 昨天回去路上我还想起你那个表情，挺有意思的
2. 昨天认识得有点快，补一句，跟你聊天还挺轻松
3. 你昨天那个点子我后来越想越好笑

最稳的是第 3 条。
发完别补第二句，等她接。
```

### 她回得冷

用户：

```text
我说周末去喝咖啡吗，她回“再看吧”
```

Skill：

```text
阶段：邀约试探
信号：黄灯偏红
目标：保住体面，不追加压力

可直接发：
1. 行，那你忙完再说
2. 好，等你哪天想出门了再抓你
3. 没事，你先忙你的

最稳的是第 1 条。
这时候别追问“那你什么时候有空”。
```

## 安全边界

这个 skill 不做的事很明确：

- 不帮用户操控、驯化或羞辱女性
- 不帮用户绕过明确拒绝
- 不教用户靠忽冷忽热制造依赖
- 不把关系当征服游戏

如果用户想要的是“更有主导感、更有吸引力”，skill 会用这些健康替代：
- 主导感
- 边界感
- 筛选感
- 节奏感
- 体面止损
